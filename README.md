# dev-env

Personal development environment for macOS.

## Setup

```bash
# Install Homebrew tools
./run

# Symlink configs to $HOME
./dev-env
```

## What's included

- **Neovim** — LSP, Treesitter, Telescope, Harpoon, Rose Pine theme
- **Tmux** — Alt-key navigation, status bar at top
- **Ghostty** — Terminal emulator
- **AeroSpace** — Tiling window manager with named workspaces
- **Starship** — Shell prompt
- **OpenCode** — Claude Code IDE config
